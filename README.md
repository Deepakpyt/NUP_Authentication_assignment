# Prequisites

* Nodejs
* MongoDB


### Libraries

1. express
2. mongoose
3. dotenv
4. jsonwebtoken
5. bcrypt
6. joi
7. nodemon


## Environment Variables Needed:-

* MONGO_DATABASE
* TOKEN_SECRET

## Run Project
```bash
npm install
npm start
```
## URLs
* POST /users/register -> Register a new user
* POST /users/login -> Login to get the AuthToken
* GET /users/profile -> Use your AuthToken to access your Profile
